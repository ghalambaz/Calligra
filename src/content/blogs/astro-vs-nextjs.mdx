# Astro vs. Next.js

Both are great. The choice depends on what you're building.

## Next.js Strengths

- Server-side rendering (SSR)
- API routes built-in
- Strong React ecosystem
- Vercel integration

## Astro Strengths

- Zero JS by default
- Multi-framework support
- Ideal for content sites
- Simpler mental model

## The Verdict

Build a **marketing site or blog** → Astro. Build a **web app with auth and dynamic data** → Next.js.
