# Introduction to Astro 🚀

Astro is gaining popularity fast. Here's why it matters.

## The Core Idea

Astro ships **zero JavaScript** by default. Components render to static HTML at build time. You opt-in to JavaScript only where needed.

## Islands Architecture

Interactive "islands" of JavaScript sit in a sea of static HTML. Each island hydrates independently.

```astro
---
import Counter from './Counter.jsx';
---
<h1>Static heading</h1>
<Counter client:visible />
```

## Why Developers Love It

- Blazing-fast Lighthouse scores
- Use React, Vue, Svelte — or none
- File-based routing
- MDX support out of the box
