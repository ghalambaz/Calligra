# Reusable Components in Astro

Astro components are `.astro` files. They're like HTML templates with a script fence at the top.

```astro
---
// Component Script (runs at build time)
const { title, description } = Astro.props;
---
<div class="card">
  <h2>{title}</h2>
  <p>{description}</p>
</div>
```

## Using Framework Components

You can drop in React, Vue, or Svelte components:

```astro
---
import ReactButton from './Button.jsx';
import VueModal from './Modal.vue';
---
<ReactButton client:load label="Click me" />
<VueModal client:visible />
```

## Tips

- Keep components small and focused
- Use props for configuration
- Use slots for content injection
