# CSS Grid Mastery

Grid makes complex layouts declarative and maintainable.

## Auto-fit Columns

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1.5rem;
}
```

Responsive columns — no media queries needed.

## Subgrid for Alignment

```css
.card {
  display: grid;
  grid-row: span 3;
  grid-template-rows: subgrid;
}
```

All cards align their internals across columns.

## Named Areas

```css
.layout {
  grid-template-areas:
    "nav nav"
    "sidebar content"
    "footer footer";
}
```

Self-documenting layouts at a glance.
